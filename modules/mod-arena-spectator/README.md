# Arena Spectator Module

This module provides arena spectator functionality for AzerothCore, allowing players to spectate ongoing arena matches.

## Features

- Spectate ongoing arena matches
- Real-time updates of player stats, cooldowns, and auras
- Watch command to switch between players
- Addon integration for enhanced spectating experience

## Commands

- `.spect spectate <player>` - Start spectating an arena match
- `.spect watch <player>` - Switch to watching a different player in the same arena
- `.spect leave` - Leave spectator mode
- `.spect reset` - Reset spectator addon data
- `.spect version <version>` - Check addon version compatibility

## Requirements

- Compatible with Arena Spectator addon version 27+

## Configuration

This module is built into the core and does not require separate configuration.
